# Require.sh

Shell script like NodeJS to use require to load other shell

### Installation

##### via npm
```bash
$ npm install -g require.sh
```

##### via git
```bash
$ git clone https://github.com/yorkie/require.sh.git
```

### Usage

Just like following in your script:

```bash
require example.sh
```

### Test

```bash
$ npm test
```
